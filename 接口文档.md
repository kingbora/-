## 毕业设计接口文档##

#### 1、用户登陆

- **请求URL**
> [/user/login](http://localhost:8080/Kingbora/user/login)

- **请求方式** 
> **POST**

- **请求参数**
>
 | 请求参数      |     参数类型 |   参数说明   |
| :-------- | :--------| :------ |
| account | String |  账号 |
| password |   String |  密码 |

- **返回参数**
>
| 返回参数      |     参数类型 |   参数说明   |
| :-------- | :--------| :------ |
| data |   Object |  返回数据 |
| status |   Integer |  执行结果code **200.登陆成功 401.验证失败 403.用户不存在**|

- **返回示例**
>    
```json
{
  "created_at": "2017-03-21 11:01:26.0",
  "password": "520kwb",
  "phone": "17321345207",
  "portrait": "default.jpg",
  "userId": 1,
  "userName": "5tgljz1q",
  "userSex": 2
}
```
